## ✨ المميزات

- 🕌 **القرآن الكريم كاملاً** - جميع السور الـ 114
- 🔍 **البحث** - بحث سريع في آيات القرآن
- 🕐 **أوقات الصلاة** - حسب موقعك الجغرافي
- 🧭 **اتجاه القبلة** - بوصلة تفاعلية
- 📖 **التفسير** - تفسير الآيات
- 🎧 **الاستماع** - أشهر القراء
- 📱 **تصميم متجاوب** - جميع أحجام الشاشات

## 🔧 المتطلبات

- Android 7.0+ (API 24)
- الحجم: ~4.5 MB
